# quick-quote-generator
A simple browser-based quote generator with export functionality.

## Live Demo
https://david-neum.github.io/quick-quote-generator/

## Download
Available via Releases

## Run locally
Clone the repository and open index.html in your browser

## Features
- Add and remove quote items dynamically
- Automatic total calculation
- Currency formatting (currently $ USD, € EUR, £ GBP, no conversion!)
- Live list updates
- Export quote as downloadable text file
